Como publicar com dados reais

1
Configure a API Key do Google Sheets
Acesse console.cloud.google.com → APIs & Services → Credentials → Criar chave de API. Ative a Google Sheets API.
2
Torne a planilha pública (somente leitura)
Na planilha, clique em Compartilhar → Qualquer pessoa com o link → Leitor. Isso permite que o dashboard leia os dados sem login.
4
Hospede gratuitamente no GitHub Pages
Crie um repositório no GitHub → faça upload do arquivo HTML → vá em Settings → Pages → Source: main branch. Seu dashboard estará em seu-usuario.github.io/seu-repo em minutos.
5
Alternativas de hospedagem rápida
Netlify Drop (arraste o arquivo em netlify.com/drop) ou Vercel CLI (npx vercel). Ambos gratuitos e sem configuração.
